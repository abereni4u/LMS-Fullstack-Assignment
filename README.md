# LMS-Fullstack-Assignment
A simple Learning Management System. Django REST Framework backend, React + TypeScript frontend. Instructors create courses and chapters (rich text via Plate.js) and set chapter visibility; students join courses and read public chapters.

## Run
Backend: `cd backend`, activate venv, `pip install -r requirements.txt`, `python manage.py migrate`, `python manage.py runserver`
Frontend: `cd frontend`, `npm install`, `npm run dev`
